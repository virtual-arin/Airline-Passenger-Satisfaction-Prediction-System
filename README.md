# Airline-Passenger-Satisfaction-Prediction System 👦👧

## ✈️ Business Domain
Aviation

## 🤔 Problem Statement

* Airlines today operate in a highly competitive environment where customer experience directly impacts loyalty, revenue, and brand reputation. Despite collecting extensive passenger feedback and operational data, it is challenging to proactively identify dissatisfied customers and understand the key drivers behind their experience.

* This project focuses on building a machine learning classification system that predicts whether an airline passenger is Satisfied or Neutral/Dissatisfied based on demographic information, flight details, and service quality ratings. The goal is to transform historical passenger data into actionable insights that can support data-driven improvements in airline service operations.

## 🧠 Objective

The primary objective of this project is to develop a robust binary classification model that:
* Predicts passenger satisfaction levels (Satisfied vs Neutral/Dissatisfied)
* Identifies the most influential factors affecting customer experience
* Helps airlines proactively detect dissatisfaction patterns
* Enables data-driven improvements in service quality and operational efficiency
* Satisfied : 1
* Dissatisfied : 0
